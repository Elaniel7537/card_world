# Card World

El aplicativo esta desarrollado en Next.js.

## Implementado:

- React.
- Typescript.
- Atomic Desing.
- Ant Desing. https://ant.design/.
- Mobil Firts.
- Fecth.
- Animate.

## Iniciar Proyecto

Pasos para desplegar en servidor local.

```bash
yarn install

yarn dev - ambiente desarrollo

yarn build - build

yarn start - servidor
```

Abrir http://localhost:3000 resultados locales.
